# TodoHeap APP

# 开发环境

## 后端
- 安装 Python3
- 安装 uv
- 安装项目依赖
    ```bash
    cd backend
    uv sync
    ```
- 运行
    ```bash
    uv run main.py
    ```
- 访问: http://localhost:8000/

## 前端(Vue版):

- 安装 Node.js
- 安装项目依赖
    ```bash
    cd frontend-vue
    npm install
    ```
- 运行
    ```bash
    npm run dev
    ```
- 访问: http://localhost:3000

- 构建
    ```bash
    npm run build
    ```

- 提交后可访问: https://sherlocknieh.github.io/TodoHeap/

## 前端(Next.js版):

- 安装 Node.js
- 安装项目依赖
    ```bash
    cd frontend-nextjs
    npm install
    ```
- 运行
