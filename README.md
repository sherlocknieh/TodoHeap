# TodoHeap APP

# 开发环境

## 后端
- 安装 Python3
- 安装 uv
- 安装项目依赖
    ```bash
    cd backend
    uv sync
    ```
- 运行
    ```bash
    cd backend
    uv run main.py
    ```
- 访问: http://localhost:8000/

## 前端(Vue版):

- 安装 Node.js
- 安装项目依赖
    ```bash
    cd frontend-vue
    npm install
    ```
- 运行
    ```bash
    cd frontend-vue
    npm run dev
    ```
- 本地调试: http://localhost:5173/TodoHeap/vue/
- 在线访问: https://sherlocknieh.github.io/TodoHeap/vue/
