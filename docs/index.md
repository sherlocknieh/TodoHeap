# TodoHeap 文档

欢迎来到 **TodoHeap** - 一个 AI 驱动的智能任务管理应用！

## 项目简介

TodoHeap 基于普通 TodoList 应用的基础上，添加了 AI 自动分解任务的功能：

- 🤖 **AI 自动分解任务** - 借助AI技术，把大任务自动分解细化为能快速完成的小任务
- 📋 **思维导图式任务管理** - 拖动节点管理任务，高效地整理思路
- 🎯 **智能堆排序** - 以堆视图展示任务, 最优先的任务置于最顶端，解决选择困难症
