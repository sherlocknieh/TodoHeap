# TodoHeap 文档

欢迎来到 **TodoHeap** - 一个 AI 驱动的智能任务管理应用！

## 项目简介

TodoHeap 基于普通 TodoList 应用的基础上，添加了 AI 自动分解任务的功能：

- 🤖 **AI 自动分解** - 把大任务自动分解细化为能快速完成的小任务
- 📋 **任务管理** - 减轻任务管理压力，提高工作效率
- 🎯 **智能排序** - 找出最优先的任务置于最顶端，省去在多个平行任务中做选择的精力
