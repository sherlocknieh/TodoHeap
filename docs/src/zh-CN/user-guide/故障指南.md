# 用户文档补充

高级功能和故障排查

## 高级功能

### 自定义标签

为任务添加自定义标签，便于分类和搜索。

### 优先级设置

支持四个优先级：
- 🔴 紧急
- 🟠 高
- 🟡 中  
- 🟢 低

### 任务统计

在仪表板上查看：
- 本周完成数
- 平均完成时间
- 待办任务数

## 故障排查

### 数据没有同步

1. 检查网络连接
2. 刷新页面
3. 检查浏览器控制台是否有错误

### 无法分解任务

可能原因：
- AI 额度已用尽，需要续费
- 网络连接不稳定
- 任务描述不清晰，可以补充更多细节

### 登录失败

1. 确保邮箱地址正确
2. 尝试重置密码
3. 清除浏览器缓存后重试
